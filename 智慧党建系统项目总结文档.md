# 智慧党建系统项目总结文档

## 📋 项目概述

智慧党建系统是一个基于Spring Boot + Vue3 + 微信小程序的现代化党建管理平台，旨在通过数字化手段提升党建工作效率，实现党员管理、学习、活动、积分等功能的全面信息化。

### 🎯 项目目标
- 实现党员信息数字化管理
- 提供在线学习资源和进度跟踪
- 建立完善的积分激励体系
- 支持多层级组织架构管理
- 提供便捷的移动端体验

### 🏗️ 技术架构
- **后端**: Spring Boot 3.5.4 + MyBatis + MySQL
- **管理端**: Vue3 + Element Plus + Pinia
- **用户端**: 微信小程序 (uni-app)
- **数据库**: MySQL 8.0
- **部署**: Docker + Nginx

private void awardTaskPoints(Long taskId, List<TaskAssignment> assignments, 
                           Integer taskPoints, String taskTitle) {
    for (TaskAssignment assignment : assignments) {
        boolean alreadyAwarded = checkIfTaskPointsAlreadyAwarded(
            assignment.getUserId(), taskId);
        if (!alreadyAwarded) {
            Points points = new Points();
            points.setUserId(assignment.getUserId());
            points.setPoints(taskPoints);
            points.setDescription("完成任务：" + taskTitle + " taskId:" + taskId);
            pointsMapper.insert(points);
        }
    }
}

public Result<Map<String, String>> uploadFile(@RequestParam("file") MultipartFile file) {
    try {

@PostMapping("/progress")
public Result<Void> updateProgress(@RequestBody VideoProgressDTO progressDTO) {
    try {
        // 计算观看百分比
        double percentage = (double) progressDTO.getCurrentTime() / 
                          progressDTO.getTotalDuration() * 100;
        
        // 更新或插入进度记录
        VideoWatchProgress progress = new VideoWatchProgress();
        progress.setUserId(progressDTO.getUserId());
        progress.setResourceId(progressDTO.getResourceId());
        progress.setCurrentTime(progressDTO.getCurrentTime());
        progress.setTotalDuration(progressDTO.getTotalDuration());
        progress.setWatchPercentage(BigDecimal.valueOf(percentage));
        progress.setLastWatchTime(LocalDateTime.now());
        
        videoWatchProgressMapper.insertOrUpdate(progress);
        
        return Result.success();
    } catch (Exception e) {
        return Result.error("进度更新失败");
    }
}
