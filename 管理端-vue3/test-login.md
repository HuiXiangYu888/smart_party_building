## 🧪 测试步骤

### 1. 启动项目
```bash
npm run dev
```

### 2. 访问登录页面
打开浏览器访问：`http://localhost:3000/login`

### 3. 测试登录
- **用户名**：输入任意3-20个字符（如：admin）
- **密码**：输入任意6-20个字符（如：123456）
- 点击"登录"按钮

### 4. 测试功能模块

#### 4.1 党建数据

## 🔍 功能特性

### 账号管理特性
- **身份权限**：
  - 超级管理员：可管理所有数据
  - 党支部管理员：只能查看对应党支部数据
  - 普通管理员：基础管理权限

- **数据查看**：
  - 点击"查看数据"可查看对应党支部的统计信息
  - 包括党员数量、申请数量、活动数量等

- **账号操作**：
  - 支持添加、编辑、删除管理员
  - 可选择身份和所属党支部
  - 可设置账号启用/禁用状态

## 🚨 常见问题

### 1. 登录后不跳转
- 检查路由配置是否正确
- 确认 `router.push('/dashboard')` 执行成功

### 2. 侧边栏菜单不显示
- 检查Layout组件是否正确加载
- 确认菜单路径配置正确

### 3. 账号管理功能异常
- 检查表单验证规则
- 确认数据绑定正确
