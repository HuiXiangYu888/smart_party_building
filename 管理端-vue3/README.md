# 智慧党建管理端 Vue3 版本

这是一个基于 Vue3 + Element Plus 的智慧党建管理端系统，可以在 HBuilderX 中运行。

## 项目特性

- 🚀 基于 Vue3 + Vite 构建
- 🎨 使用 Element Plus UI 组件库
- 📱 响应式设计，支持移动端
- 🔐 完整的登录认证系统
- 📊 数据可视化仪表盘
- 🎯 党员管理、申请审核、公告管理等功能
- 📚 学习资源管理和进度跟踪
- ✅ 任务管理和进度监控

## 技术栈

- **前端框架**: Vue 3.3.4
- **构建工具**: Vite 4.4.5
- **UI 组件库**: Element Plus 2.3.8
- **状态管理**: Pinia 2.1.6
- **路由管理**: Vue Router 4.2.4
- **HTTP 客户端**: Axios 1.4.0
- **样式预处理**: Sass 1.64.1

## 在 HBuilderX 中运行

### 1. 打开项目

1. 打开 HBuilderX
2. 选择 `文件` -> `打开目录`
3. 选择 `管理端-vue3` 文件夹

### 2. 安装依赖

在 HBuilderX 的终端中运行：

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

项目将在 `http://localhost:3000` 启动。

### 4. 构建生产版本

```bash
npm run build
```

构建后的文件将生成在 `dist` 目录中。

## 开发说明

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

### 开发命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 代码规范

- 使用 Vue3 Composition API
- 使用 TypeScript 类型注解（可选）
- 遵循 ESLint 规范
- 使用 Prettier 格式化代码

## 部署说明

### 1. 构建项目

```bash
npm run build
```

### 2. 部署到服务器

将 `dist` 目录中的文件部署到 Web 服务器即可。

### 3. 配置代理

在生产环境中，需要配置 API 代理指向后端服务。

## 注意事项

1. 确保后端 API 服务正常运行
2. 检查 API 接口地址配置是否正确
3. 确保网络连接正常
4. 如遇到跨域问题，请检查代理配置

## 常见问题

### Q: 项目无法启动？
A: 请检查 Node.js 版本是否符合要求，并确保所有依赖都已正确安装。

### Q: API 请求失败？
A: 请检查后端服务是否正常运行，以及 API 地址配置是否正确。

### Q: 样式显示异常？
A: 请确保 Element Plus 样式文件已正确引入。
